# Markdown to HTML Converter
Converts markdown to HTML based on the "Basic Syntax" section at (https://www.markdownguide.org/cheat-sheet/):
- headings (3 levels)
- bold
- italic
- blockquote
- orderer list (only one line per list item)
- unordered list
- code
- horizontal rule
- link
- image

## Tools
PHP8 / JavaScript / CSS3 /HTML5
